# Design system notes — extracted from index.html

Reference these for deeper.html so the page reads as the same designer's hand.

## Palette (CSS custom properties)

```
--paper:       #f6f2eb   /* main background */
--paper-2:     #efe9df   /* card backgrounds, table header bg */
--paper-3:     #e6dfd0   /* tertiary surfaces */
--ink:         #1a1a1a   /* primary text */
--ink-2:       #3a3a3a   /* body text */
--ink-3:       #6b6258   /* metadata, captions */
--ink-4:       #9a9082   /* faint metadata */
--rule:        #d8d0c2   /* solid borders */
--rule-soft:   #e6dfd0   /* subtle borders */
--accent:      #8a3a2a   /* terracotta — eyebrows, links, hero gradients */
--accent-soft: #c97a5a
--pass:        #3a6a4a   /* green for pass states */
--fail:        #a05050   /* dusty red for fail states */
```

## Typography

- Body sans: `'Inter', system-ui, sans-serif` — 15px / 1.55 line-height baseline.
- Serif display: `'Spectral', Georgia, serif` — used for h1/h2/h3, lede, italic asides.
- Italic Spectral 300 weight = reflective voice (subtitle, footer frame, italic closers).
- Tabular numbers: `font-feature-settings: 'tnum' 1; font-variant-numeric: tabular-nums;` on `.mono` and number cells.
- Eyebrow: 11px / 600 / 0.12em letter-spacing / uppercase / `--accent` color.
- `.small`: 11.5px / `--ink-3`.

Google Fonts link:
```html
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&family=Spectral:ital,wght@0,300;0,400;0,500;0,600;1,400&display=swap" rel="stylesheet" />
```

## Layout patterns

- `.container { max-width: 1180px; margin: 0 auto; padding: 0 32px; }`
- Sections are separated by `border-bottom: 1px solid var(--rule);` and padded `60–80px` top/bottom.
- Hero: `clamp(48px, 8vw, 96px)` Spectral 400 h1, italic Spectral subtitle, lede in serif 17px.
- Background ornament for hero & contact: dual radial gradients with accent + a complementary cool tone (`rgba(44, 95, 124, 0.05)`).
- Cards: bordered, `border: 1px solid var(--rule);`, sometimes `border-left: 3px solid var(--accent)` for callouts (see `.invitation-cta`).

## Aesthetic vocabulary

- Atelier / paper-feeling. Not glossy.
- Dotted underlines for links inside prose (`text-decoration: underline dotted; text-underline-offset: 3px;`).
- `border-left: 2–3px solid var(--accent)` is the recurring callout device.
- Headings sit close to descenders; tight line-heights (1.0–1.2) on display sizes.
- Italic closers / reflective sentences in Spectral italic 300 — used in hero subtitle and footer frame.
- Section eyebrows in `.accent` color, all caps, letter-spaced.

## Deeper.html — additional patterns to introduce

- `.methodology-point` and `.tension-pair` callouts can extend the `.invitation-cta` pattern (paper-2 fill + accent left rule).
- Case studies want roomy max-width (`max-width: 64ch`) measure for the body, with sourced numbers landing in-prose, never as bullets.
- Closing italic paragraph of each case study uses Spectral italic 300, slightly larger (17–18px) and `--ink-3` so it reads as voice-shift.
- Sources list is a grouped definition-list: dimension heading (Inter caps eyebrow style) followed by source name + cited URL + "Used for: <regions>" small line.
- Maintain mobile breakpoint: at ≤780px collapse two-column blocks to single column with 32px gap (see existing `.contact .col` rule).
